# LeetCode Practice Journal

A simple LeetCode tracker I built to keep my DSA practice organized.

I wanted something more visual than a spreadsheet or a normal checklist, so I made it look like a practice journal. Problems are grouped by DSA topic and I can check them off as I work through them.

## What it does

- Tracks solved problems
- Organizes problems by DSA topic
- Shows overall progress
- Keeps track of Easy, Medium and Hard problems
- Lets me star problems I want to come back to
- Opens problems directly on LeetCode
- Keeps a recent activity log
- Saves progress in the browser using localStorage

## Roadmap

The journal currently covers:

- Arrays & Hashing
- Two Pointers
- Stacks
- Binary Search
- Sliding Window
- Linked Lists
- Trees
- Heap / Priority Queue
- Backtracking
- Tries
- Graphs
- Advanced Graphs
- 1-D and 2-D Dynamic Programming
- Greedy
- Intervals
- Bit Manipulation
- Math & Geometry

## Built with

HTML, CSS and vanilla JavaScript.

I kept it frontend-only for now. Progress is stored with localStorage, so there's no account or backend needed.

## Running it locally

Clone the repo:

```bash
git clone https://github.com/saahithimukala10-gif/leetcode-practice-journal.git
cd leetcode-practice-journal
```

Then open `index.html` in your browser or run it with VS Code Live Server.
